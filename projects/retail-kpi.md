# Retail KPI Dashboard  
### Sales Performance & Profitability Analysis  

---

## Project Status

This project is currently under development.

The objective is to design a structured KPI dashboard based on a retail sales dataset, focusing on revenue trends, profitability, category performance, and regional analysis.

---

## Work in Progress

The current development phase includes:

- Data cleaning and normalization  
- Definition of core KPIs (Revenue, Margin %, Growth Rate)  
- Design of a Star Schema data model  
- SQL-based aggregations for business metrics  
- Dashboard layout planning in Power BI  

The final version will include:

- Fully developed interactive dashboard  
- Business insights derived from quantitative analysis  
- Strategic recommendations based on performance patterns  

---

## Analytical Focus

The goal of this project is not only to build a dashboard, but to demonstrate:

- Business-oriented KPI design  
- Structured problem-solving  
- Decision-support analysis  
- Clear insight communication  

Results and insights will be published soon.
