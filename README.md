# Inventory Optimization & Stockout Analysis

**Objective:**  
Analyze inventory data to identify stockout risks and improve service levels using KPI-driven analysis.

**Workflow:**  
1. **Data preparation:** SQL scripts to clean and structure sales, stock, and replenishment data.  
2. **KPI calculation:** Inventory turnover, stockout rate, service level %.  
3. **Analysis:** Python for additional data validation and forecasting.  
4. **Dashboard:** Power BI dashboard visualizing KPIs and operational insights.

**Technical Stack:**  
- PostgreSQL (SQL queries, data modeling)  
- Python (pandas, EDA)  
- Power BI (dashboard and KPI visualization)

**Status:** In development
